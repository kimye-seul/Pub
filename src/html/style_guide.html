[리스트 코멘트]

  • H.Point 회원가입은 만 14세 이상만 가입 가능합니다.
  • 네이버 간편 가입은 네이버 인증서 발급 회원만 가능합니다.

  • 필수항목에 동의하지 않으실 경우 서비스 가입이 불가합니다.
  • 선택항목에 동의하지 않으셔도 서비스 가입이 가능하나, 관련 서비스는 제공받으실 수 없습니다.

  • 온라인 결제 : APP 내 현금영수증 발행정보 등록 시, 포인트 사용분 현금영수증 자동 발행
  • 오프라인 결제 : 매장 결제 시 현금영수증 발급을 요청

  • 브랜드별 기본 적립액의 2배 즉시 적립
  • 가입 완료(승인) 익일 결제 건부터 적용

  • 단, 본 서비스는 현대백화점, 현대프리미엄아울렛, 현대시티아울렛 등
    현대백화점 오프라인 매장에서만 이용 가능합니다.
  • H.Point APP, 백화점카드데스크에서 신청가능합니다.
